# 函数
# 1 定义
- 你可以定义一个由自己想要功能的函数，以下是简单的规则：

    - 函数用 `def` 关键词定义，后接 `函数名` 和 `圆括号 ()`
    - 圆括号内可以设置参数
    - 函数体内通常会有一些注释说明参数、返回值。
    - 函数内容以冒号 : 起始，并且缩进。
    - return [表达式] 结束函数，选择性地返回一个值给调用方，不带表达式的 return 相当于返回 None。


>语法

- Python 定义函数使用 def 关键字，一般格式如下：
```python
def 函数名（参数1: 参数类型, 参数2: 参数类型, ..., *args, **kwargs）:
    '''
    函数说明
    '''
    函数体
    return # 可选
```
默认情况下，参数值和参数名称是按函数声明中定义的顺序匹配起来的。

>示例
```python
# 以下 3 种形式等价
def hello() :
    print("Hello World!")

def hello() :
    print("Hello World!")
    return 

def hello() :
    print("Hello World!")
    return None

hello()

def max(a, b):
    if a > b:
        return a
    else:
        return b
 
a = 4
b = 5
print(max(a, b))

def area(width, height):
    return width * height
 
def print_welcome(name):
    print("Welcome", name)
 
print_welcome("aaaa")
w = 4
h = 5
print("width =", w, " height =", h, " area =", area(w, h))
```

# 2 函数调用
- 定义一个函数：给了函数一个名称，指定了函数里包含的参数，和代码块结构。
>示例
```python
# 定义函数
def printme(string):
   # 打印任何传入的字符串
   print(string)
   return
 
# 调用函数
printme("我要调用用户自定义函数!")
printme("再次调用同一函数")
```

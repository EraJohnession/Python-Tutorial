
# Python 赋值运算符
以下假设变量a为10，变量b为20：

运算符|	描述	|实例
|:-- |:-- |:--|
=	|简单的赋值运算符	|c = a + b 将 a + b 的运算结果赋值为 c
+=	|加法赋值运算符	|c += a 等效于 c = c + a
-=	|减法赋值运算符	|c -= a 等效于 c = c - a
*=	|乘法赋值运算符	|c *= a 等效于 c = c * a
/=	|除法赋值运算符	|c /= a 等效于 c = c / a
%=	|取模赋值运算符	|c %= a 等效于 c = c % a
**=	|幂赋值运算符	|c **= a 等效于 c = c ** a
//=	|取整除赋值运算符	|c //= a 等效于 c = c // a




>花式赋值
```python
a, b, c = [1, 2, 3]
print(a, b, c)   # 输出：1 2 3
a, b, c = (1, 2, 3)
print(a, b, c)   # 输出：1 2 3
a, b, c = "123"
print(a, b, c)   # 输出：1 2 3
a, b, *c = [1, 2, 3, 4]
print(a, b, c)   # 输出：1 2 [3, 4]
a, *b, c = [1, 2, 3, 4]
print(a, b, c)   # 输出：1 [2, 3] 4
```

>示例
```python
a = 21
b = 10
c = 0
 
c = a + b
print ("1 - c 的值为：", c)
 
c += a
print ("2 - c 的值为：", c)
 
c *= a
print ("3 - c 的值为：", c)
 
c /= a 
print ("4 - c 的值为：", c)
 
c = 2
c %= a
print ("5 - c 的值为：", c)
 
c **= a
print ("6 - c 的值为：", c)
 
c //= a
print ("7 - c 的值为：", c)
```
